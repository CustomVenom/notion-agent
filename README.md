# Notion Workspace Agent for VS Code

Query your Notion workspace directly from VS Code chat with `@roadmap` - zero token consumption.

## Quick Start
1. Install the extension
2. Configure Notion token in VS Code settings
3. Use `@roadmap what's P0 today?` in VS Code chat

See [installation guide](link to Notion doc) for full setup.

## Token Savings
- Current: 85k tokens per query (~$0.17)
- With @roadmap: 0 tokens ($0)
- Monthly savings: $40-60 at scale
